# dxspider basic web
Simple php page for dxspider cluster database.

Configure the connection to the MySQL database, edit local/DXVars.pm:



$dsn = "dbi:mysql:YOUR-DB-NAME:localhost:3306";

$dbuser = "YOUR-DB-USER";

$dbpass = "YOUR-DB-PASSWORD";
